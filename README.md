# Home Infrastructure

In questo repo ho raccolto una serie di ansible che ho creato per mettere su un laboratorio domestico usando una serie di PC che avevo dentro casa senza più un uso ben definito. 

L'obiettivo (per ora) è quello di realizzare una VPN per l'accesso remoto e un cluster kubernates su cui hostare le applicazioni. 

Chiaramente questa configurazione non riguarda delle macchine reali ma allo strato più basso trova un cluster proxmox equipaggiato con una serie di macchine Fedora CoreOS.
